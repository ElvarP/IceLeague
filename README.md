## Hvað er þetta?

Þetta er kóðinn fyrir vefsíðuna http://iceleague.is

Það var ekki til nein góð leið til að finna alla íslenska League Of Legends spilara þannig ég ákvað að búa til þessa síðu.

Þetta er fyrsta "alvöru" síðan sem ég hef gert (Líka í fyrsta skiptið sem ég hef unnið með php fyrir utan að installa wordpress nokkrum sinnum) og fyrsta skipti sem ég hef unnið með Github þannig ég vona að ég hafi gert allt rétt.

Ef þú sérð eitthvað sem má betrumbæta (Sem þú munt eflaust gera) þá máttu endilega gera pull request!

Síðan notar http://getbootstrap.com/, http://datatables.net og https://github.com/LeaguePHP/LeagueWrap
